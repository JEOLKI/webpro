**작성팁
	- 의사 클래스(pseudo-class, 가상클래스):  
	- 클래스가 정의된 것처럼 간주.  ( : )으로사용
	- 상태나 구조에 의해서 선택이 이루어진다 
	
	상태
	- a:link    => 처음상태
	- a:visited => 한번 방문한 상태
	- a:hover	=> 마우스가 올려있는 상태
	- a:active	=> 마우스를 누르고 있는 상태
